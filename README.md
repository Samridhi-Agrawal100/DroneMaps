## Installation

### Clone the repository and make your own branch

```bash
git clone <your-repo-url>
cd DroneMaps

git checkout -b <their-name>-feature
git push -u origin <their-name>-feature
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser at:

http://localhost:5173
